Creating Posts
---

1. Run `raco frog -n "TITLE"`
2. Edit the newly-generated markdown file
3. Rebuild the blog with `raco frog -b`; open `index.html` to view
4. Remove the `DRAFT` tag
5. Commit & push your blog post
